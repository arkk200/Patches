# Patches Game Rules

**Patches**는 보드판을 겹치지 않는 직사각형 영역(Patches)으로 나누는 퍼즐 게임입니다. 이 게임은 **Shikaku(시카쿠)**라는 이름으로도 알려져 있습니다.

## Objective (목표)
N x N 크기의 그리드 보드를 여러 개의 직사각형 영역으로 빈틈없이 나누는 것입니다.

## Rules (규칙)
1. **숫자와 영역**: 모든 직사각형 영역은 반드시 **단 하나의 숫자**만을 포함해야 합니다.
2. **영역의 넓이**: 직사각형 영역의 칸 수(넓이)는 그 영역 안에 포함된 **숫자**와 정확히 일치해야 합니다.
3. **직사각형 형태**: 모든 영역은 $n \times m$ ($n, m$은 정수) 형태의 **직사각형**이어야 합니다.
4. **중복 금지**: 영역들은 서로 겹칠 수 없습니다.
5. **완전성**: 보드의 모든 칸은 반드시 어떤 하나의 직사각형 영역에 속해야 합니다 (빈 칸이 없어야 함).

## Example (예시)
- **5**라는 숫자가 있는 칸: 
  - $1 \times 5$ 또는 $5 \times 1$ 크기의 직사각형으로 확장되어야 하며, 그 영역 안에는 다른 숫자가 포함되어서는 안 됩니다.
- **8**이라는 숫자가 있는 칸:
  - $1 \times 8$, $8 \times 1$, $2 \times 4$, $4 \times 2$ 중 하나의 직사각형으로 확장되어야 합니다.

## Implementation Details
- 보드는 2차원 배열로 관리됩니다.
- 각 패치(직사각형)는 시작 좌표, 끝 좌표, 그리고 포함된 숫자 값으로 정의될 수 있습니다.
